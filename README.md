# Boombox Documentation
**Shimmy's Customs** - 3D Printed Boombox Information

## About

This repository contains assembly documentation, guides, and resources for custom 3D-printed boombox parts designed by Shimmy's Customs.

---

## Available Designs

### Apache 1800 All-In-One Insert
A 3-piece modular system designed for the Apache 1800 hard case.

- **STL Files:** [MakerWorld - Apache 1800](link-coming-soon)
- **Assembly Guide:** [1800-boombox-assembly-guide.pdf](./apache-1800/1800-boombox-assembly-guide.pdf)
- **Features:** Battery compartment, amplifier mounting, integrated dual-lock tabs

### Apache 2800 All-In-One Insert
A 3-piece modular system designed for the Apache 2800 hard case.

- **STL Files:** [MakerWorld - Apache 2800](link-coming-soon)
- **Assembly Guide:** [2800-boombox-assembly-guide.pdf](./apache-2800/2800-boombox-assembly-guide.pdf)
- **Features:** Larger battery capacity, enhanced component organization
